公开摘要

这份招聘需求以完成业务系统为目标，整理后端、前端、UI、移动端、uni-app 与测试岗位的职责和任职要求。薪资、人数等内部字段在公开页面中仅作为能力配置参考，不作为招聘承诺。

面试评分口径

评分口径
60 分完成技术面评估，但暂不建议优先推进。
80 分可用，定位中级，可进入录用流程。
90 分特别优秀，定位高级，建议优先推进。

PHP 后端工程师

  • 负责互联网产品和核心业务系统的后端架构设计、开发与迭代优化。
  • 独立完成系统 API 接口设计、开发和技术文档编写。
  • 参与现有系统维护、代码重构和性能调优，保障线上稳定性。
  • 熟悉 PHP、Laravel/Lumen、MySQL、Redis/MongoDB、消息队列和 Git 工作流。
  • 具备线上故障定位能力，包括 Trace、日志、CPU/IO 等。

前端与 uni-app

  • 负责核心业务系统前端架构、开发与维护，对接 RESTful API 和数据交互。
  • 覆盖 PC、H5、WebApp、小程序或 App 等多端场景，关注弱网和低端设备体验。
  • 开发可复用业务组件，如复杂表单、表格、可视化组件。
  • 熟悉 Vue2/Vue3、TypeScript、Vite、Pinia、Vant、Element UI 或 Ant Design Vue。
  • uni-app 岗位强调一套代码多端编译、移动端体验优化和跨端协作能力。

UI 与移动端

  • UI 设计师负责 App、小程序、Web、H5 的整体界面设计和视觉规范维护。
  • 需要具备交互设计思维，能完成中高保真原型，并与前端做设计还原走查。
  • Android 工程师关注 Kotlin/Java、Jetpack、组件化、性能优化、网络通信和数据存储。
  • 跨端经验如 Flutter、React Native、uni-app 或复杂 App 性能调优经验可作为加分项。
招聘不是“补人头”，而是补齐业务系统交付链路里的关键能力。

测试工程师

  • 负责功能测试、接口测试、性能测试和自动化测试，保障产品质量。
  • 能独立编写测试计划、测试用例和测试报告，并跟踪缺陷修复。
  • 熟悉 JMeter、Postman、Swagger、SQL、Linux、Docker、Jenkins 等工具或环境。
  • 具备自动化框架搭建能力，推动测试左移和 CI/CD 中的自动化测试执行。

统一协作要求

各岗位都需要具备良好的技术文档能力、代码规范意识、团队协作意识，以及使用 AI 辅助工具提升研发效率的能力。真正可持续的团队配置，应同时覆盖架构、开发、设计、测试、部署与问题排查。
