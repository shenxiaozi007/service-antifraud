# 防诈助手需求文档 PRD

## 1. 项目概述

### 1.1 产品名称

暂定：防诈助手

可选名称：

- 防诈雷达
- 安心看听
- 帮看帮听
- 家人防骗助手

本文后续统一使用“防诈助手”。

### 1.2 产品定位

防诈助手是一款面向普通用户，尤其是中老年人、投资新手和其家属的反诈风险识别工具。用户可以上传宣传截图、聊天记录、合同图片、付款页面，或录制销售沟通、电话外放、线下讲解内容，系统通过 OCR、语音识别、风险规则库和大模型分析，输出风险等级、风险原因、关键证据和建议动作。

### 1.3 核心价值

- 帮用户快速判断宣传内容是否可信。
- 帮用户识别投资诈骗、保本高收益、冒充机构、内部消息、诱导转账等风险信号。
- 用通俗语言输出报告，便于用户自己理解，也便于分享给家人。
- 保存历史记录，形成沟通、劝阻或投诉时的参考材料。

### 1.4 产品形态

MVP 首选微信小程序。

原因：

- 使用门槛低，适合被家人转发和快速打开。
- 拍照、相册、录音、微信支付、分享能力完整。
- 不需要用户安装独立 App。

### 1.5 重要限制

微信小程序无法直接监听系统电话，也无法在后台持续录制通话。MVP 的“帮您听”应定义为：

- 用户主动打开录音；
- 电话外放或将设备靠近对话声源；
- 录音结束后生成分析报告；
- 后续可升级为录音过程中的分段实时提醒。

产品不得宣传“百分百识别诈骗”“保证防止被骗”。所有结论应使用“疑似”“存在风险信号”“建议核实”等审慎表达。

## 2. 用户与场景

### 2.1 目标用户

| 用户类型 | 特征 | 核心需求 |
| --- | --- | --- |
| 中老年用户 | 容易接触电话推销、理财课程、保健品、投资项目 | 快速判断真假，获得明确提醒 |
| 投资新手 | 缺少金融常识，容易被高收益吸引 | 识别风险话术，避免冲动付款 |
| 子女/家属 | 担心父母被骗 | 帮父母检查材料，分享风险报告 |
| 社区/反诈工作人员 | 需要辅助判断宣传材料 | 快速生成风险说明 |

### 2.2 高频场景

1. 用户收到“内部投资项目”“稳赚不赔”宣传图，上传截图分析。
2. 用户加入股票群，老师推荐“内部消息股”，上传聊天截图分析。
3. 用户接到推销电话，打开小程序录音，结束后查看风险报告。
4. 用户参加线下会议，录下讲解内容，判断是否存在虚假宣传。
5. 子女让父母把可疑材料发来，自己用小程序分析后把报告转回给父母。

### 2.3 用户痛点

- 不懂金融、医疗、养老、保健品等专业术语，难以判断真假。
- 对方话术有压迫感，例如名额有限、今天必须交钱、不要告诉家人。
- 即使怀疑有问题，也说不清具体哪里不对。
- 缺少能给家人看的简洁证据。

## 3. 产品目标

### 3.1 MVP 目标

在 4-6 周内完成一个可上线测试的小程序 MVP，覆盖：

- 图片上传分析；
- 录音上传分析；
- 风险报告；
- 历史记录；
- 点数消耗；
- 基础管理后台。

### 3.2 验证问题

- 用户是否愿意上传截图或录音进行风险分析？
- 用户是否能理解风险报告？
- 风险报告是否能帮助用户暂停转账、付款或继续配合？
- 用户是否愿意为单次分析或点数包付费？
- 家人分享是否是主要增长路径？

### 3.3 核心指标

| 指标 | MVP 参考目标 |
| --- | --- |
| 首页到分析发起转化率 | >= 35% |
| 分析完成率 | >= 85% |
| 报告页分享率 | >= 15% |
| 高风险报告后再次使用率 | >= 20% |
| 用户主动删除文件功能使用可达性 | 100% 可找到 |
| 分析失败率 | <= 5% |

## 4. 功能范围

### 4.1 MVP 必做

#### 首页

- 展示产品标题、副标题；
- 展示“帮您看”入口；
- 展示“帮您听”入口；
- 展示点数价格；
- 展示底部导航：首页、历史记录、我的。

#### 帮您看

- 支持拍照；
- 支持从相册选择图片；
- 支持最多上传 3 张图片；
- 上传前展示隐私提示；
- 上传后进入分析中状态；
- 生成图片风险报告；
- 消耗点数。

#### 帮您听

- 展示录音前说明；
- 支持开始录音、暂停、继续、结束；
- 展示录音时长；
- 录音结束后上传并分析；
- 生成录音风险报告；
- 按分钟消耗点数，不足一分钟按一分钟计算。

#### 风险报告

- 风险等级；
- 一句话结论；
- 主要风险类型；
- 风险说明；
- 关键证据原文；
- 建议动作；
- 分析时间；
- 图片数量或录音时长；
- 分享给家人；
- 保存历史记录。

#### 历史记录

- 查看全部记录；
- 按类型筛选：图片、录音；
- 按风险等级筛选：高风险、极高风险；
- 进入报告详情；
- 删除记录。

#### 我的

- 展示用户信息；
- 展示点数余额；
- 充值入口；
- 消耗记录；
- 用户协议；
- 隐私政策；
- 联系客服；
- 删除账号或清除数据入口。

#### 管理后台

- 用户列表；
- 分析记录列表；
- 报告详情；
- 文件列表；
- 点数流水；
- 风险规则管理；
- 敏感词管理；
- 分析失败重试。

### 4.2 MVP 不做

- 直接监听手机通话；
- 后台持续录音；
- 人工专家一对一审核；
- 家庭成员绑定；
- 投诉材料自动生成；
- 视频号/抖音链接解析；
- 独立 App；
- 复杂会员体系。

### 4.3 二期功能

- 录音过程中的实时分段提醒；
- 家人守护模式；
- 分享报告后家人确认；
- 高危情况一键拨打反诈热线；
- 自动生成投诉材料；
- 人工复核；
- 风险样本库；
- 社区/机构版后台。

## 5. 页面需求

### 5.1 首页

页面目标：让用户在 3 秒内理解产品，并选择“看”或“听”。

页面元素：

- 问候语：您好！
- 主标题：帮看帮听
- 副标题：帮您判断宣传是否可信，以防上当受骗
- 功能卡片：帮您看
- 功能卡片：帮您听
- 底部导航

交互：

- 点击“拍照 / 截图”进入图片上传流程；
- 点击“开始帮您听”进入录音准备页；
- 点击底部导航进入对应页面。

### 5.2 图片上传页

页面目标：引导用户上传待判断材料。

页面元素：

- 返回按钮；
- 标题：帮您看；
- 上传区域；
- 上传说明；
- 已选图片预览；
- 删除图片按钮；
- 开始分析按钮；
- 点数消耗提示。

校验：

- 未选择图片时，开始分析按钮不可用；
- 图片最多 3 张；
- 单张图片大小限制建议 10MB；
- 点数不足时引导充值。

### 5.3 图片分析中页

页面目标：降低等待焦虑。

页面元素：

- 进度状态文案；
- 安全提示；
- 取消按钮。

状态文案：

- 正在识别图片文字；
- 正在查找风险信号；
- 正在生成分析报告。

### 5.4 录音准备页

页面目标：让用户理解录音方式和限制。

页面元素：

- 返回按钮；
- 标题：帮您听；
- 录音说明；
- 计费规则；
- 余额；
- 开始录音按钮。

说明文案：

> 请将电话外放，或让手机靠近正在沟通的人声。系统会分析对话中的风险话术，仅作风险提醒参考。

### 5.5 录音中页

页面目标：完成录音采集，必要时提醒用户。

页面元素：

- 录音时长；
- 音量波形；
- 状态提示；
- 暂停/继续按钮；
- 结束并分析按钮；
- 点数预计消耗。

MVP 可不做实时转写，但应保留 UI 结构，便于二期升级。

### 5.6 报告页

页面目标：给出明确、可信、可行动的判断。

页面元素：

- 风险等级；
- 报告标题；
- 分析时间；
- 一句话结论；
- 建议动作；
- 风险点列表；
- 关键证据；
- 原始材料入口；
- 分享给家人；
- 再分析一次。

风险等级视觉：

| 风险等级 | 颜色 | 文案 |
| --- | --- | --- |
| 低风险 | 绿色 | 暂未发现明显风险 |
| 中风险 | 黄色 | 存在可疑宣传 |
| 高风险 | 橙红色 | 存在明显风险 |
| 极高风险 | 红色 | 立即停止配合 |

### 5.7 历史记录页

页面目标：让用户找到过去的判断结果。

页面元素：

- 筛选栏；
- 记录列表；
- 空状态；
- 删除入口。

列表字段：

- 风险等级；
- 分析类型；
- 报告标题；
- 分析时间；
- 费用；
- 录音时长或图片数量。

### 5.8 我的页面

页面目标：管理点数、协议、隐私和客服。

页面元素：

- 用户信息；
- 点数余额；
- 充值按钮；
- 使用记录；
- 充值记录；
- 用户协议；
- 隐私政策；
- 联系客服；
- 删除数据。

## 6. 风险分析规则

### 6.1 风险分类

| 分类 | 示例 |
| --- | --- |
| 保本高收益 | 保证收益、稳赚不赔、亏损补差 |
| 内幕消息 | 内部渠道、老师带单、主力消息 |
| 冒充机构 | 冒充证券公司、监管机构、国企项目 |
| 紧急施压 | 今天必须决定、名额有限、错过没有 |
| 私下转账 | 转到个人账户、不要走平台 |
| 隐瞒家人 | 不要告诉子女、自己决定 |
| 远程控制 | 下载会议软件、共享屏幕 |
| 敏感信息 | 验证码、银行卡、身份证、人脸识别 |
| 虚假资质 | 伪造证书、虚假备案、假牌照 |

### 6.2 风险等级判定

低风险：

- 未发现明显诈骗话术；
- 信息相对完整；
- 不涉及转账、验证码、远程控制等高危行为。

中风险：

- 存在夸大宣传；
- 收益描述不透明；
- 需要进一步核实资质。

高风险：

- 出现保本高收益；
- 出现内部消息；
- 出现冒充正规机构；
- 要求快速付款或私下转账。

极高风险：

- 要求提供验证码；
- 要求共享屏幕；
- 要求下载陌生 App；
- 要求转账到个人账户；
- 要求隐瞒家人或警方。

### 6.3 报告输出结构

```json
{
  "risk_level": "high",
  "risk_score": 86,
  "title": "疑似保本高收益投资诱导",
  "summary": "当前内容存在明显风险信号，建议暂停付款并核实对方资质。",
  "suggestions": [
    "不要继续转账或提供验证码",
    "将内容转发给家人共同确认",
    "要求对方提供可核验的官方资质"
  ],
  "risk_items": [
    {
      "category": "保本高收益",
      "description": "对方承诺收益且暗示无风险",
      "evidence": "稳赚不赔，亏了我们补"
    }
  ],
  "disclaimer": "本报告仅作风险提醒参考，不构成法律、投资或医疗建议。"
}
```

## 7. 点数与支付

### 7.1 点数规则

- 新用户赠送 30 点；
- 帮您看：20 点/次；
- 帮您听：10 点/分钟；
- 录音不足 1 分钟按 1 分钟计算；
- 分析失败不扣点或自动退回；
- 用户取消上传不扣点。

### 7.2 充值套餐

MVP 可先配置：

| 套餐 | 点数 | 价格 |
| --- | --- | --- |
| 体验包 | 100 点 | 9.9 元 |
| 常用包 | 300 点 | 25.9 元 |
| 家庭包 | 800 点 | 59.9 元 |

### 7.3 扣费时机

建议采用“预冻结 + 成功后扣除”：

1. 用户点击开始分析；
2. 检查余额；
3. 冻结预计点数；
4. 分析成功后扣除；
5. 分析失败或取消则释放冻结点数。

## 8. 合规与隐私

### 8.1 用户授权

必须获得：

- 相机权限；
- 相册权限；
- 麦克风权限；
- 文件上传和存储授权；
- 隐私政策确认。

### 8.2 数据处理

- 图片和录音用于生成风险报告；
- 默认保存历史记录；
- 用户可主动删除；
- 删除后业务侧不可再查看原始文件；
- 后台展示敏感信息时需要脱敏。

### 8.3 免责声明

报告页和协议中必须明确：

- 系统分析仅作风险提醒参考；
- 不构成法律、投资、医疗或财务建议；
- 不保证识别全部风险；
- 用户应结合官方渠道、家人意见和专业机构判断。

## 9. 验收标准

### 9.1 功能验收

- 用户可以完成图片上传并生成报告；
- 用户可以完成录音并生成报告；
- 点数可以正确扣除；
- 分析失败时不会错误扣费；
- 历史记录可查看、筛选、删除；
- 报告可分享；
- 用户可查看协议和隐私政策。

### 9.2 性能验收

- 图片分析平均完成时间 <= 15 秒；
- 1 分钟录音分析平均完成时间 <= 30 秒；
- 首页首屏加载 <= 2 秒；
- 报告页打开 <= 1 秒。

### 9.3 安全验收

- 文件访问需要鉴权；
- 用户只能查看自己的记录；
- 后台需要登录和权限控制；
- 敏感字段不明文暴露给前端；
- 支付回调验签。
